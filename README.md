# 佳佳驾到 | 洛丽塔的梦幻世界 🎀

一个梦幻可爱的洛丽塔风格音乐歌词展示页面。

## ✨ 特色功能

- 🎵 精确的 LRC 歌词同步
- 💖 梦幻的粉紫渐变背景
- ⭐ 闪烁星星装饰
- 🎀 持续飘落的可爱装饰元素
- 🌊 点击涟漪特效
- 💗 封面点击爱心爆炸特效
- 📊 音频可视化波形
- 📱 完美的手机端适配
- 🎨 可爱的圆体字体
- ⚡ 流畅的动画效果

## 🎮 交互功能

- 点击歌词跳转到对应时间
- 手机下滑自动播放音乐
- 封面图片点击特效
- 播放时封面旋转动画
- 歌词区域滑动特效

## 🚀 使用方法

1. 直接打开 `index.html` 即可
2. 或访问 GitHub Pages 在线版本

## 📁 文件说明

- `index.html` - 主页面
- `cover.jpg` - 封面图片
- `your-song.mp3` - 音乐文件
- `your-song.lrc` - 歌词文件

## 💝 技术栈

- 纯 HTML + CSS + JavaScript
- 响应式设计
- LRC 歌词解析
- Web Audio API
